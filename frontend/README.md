# Arogya AI Flutter UI

This is the UI implementation for the Arogya AI healthcare application.

## Getting Started

Since this project was generated manually (without `flutter create`), you might need to ensure your environment is set up:

1. Ensure Flutter is installed and added to your PATH.
2. Run `flutter pub get` in this directory to fetch dependencies (`google_fonts`, `lucide_icons`).
3. Run `flutter run` or open the project in Android Studio / VS Code.

## Project Structure
- `lib/theme/`: Contains color definitions matching the design.
- `lib/screens/`: Contains all UI screens organized by feature (auth, home, prescriptions, stress, profile).
- `lib/widgets/`: Contains reusable generic UI components.
